## Project Description
This project is a **fully local RAG-based chatbot** that allows users to upload documents (PDFs) or provide website URLs and chat intelligently with the content. It uses **Ollama** for running Large Language Models locally, ensuring complete privacy and no API costs.

## Features
- Upload multiple PDFs or input web URLs
- Semantic search using vector embeddings
- Interactive chat interface with conversation history
- Fully local execution using Ollama (Llama 3.2)
- No internet required after initial setup
- Responsive Streamlit web UI

## Technologies Used
- **Python**
- **Streamlit** (Frontend)
- **LangChain** (Orchestration)
- **Ollama** + Llama 3.2 (LLM)
- **FAISS** (Vector Database)
- **PyPDF** (Document Loading)
